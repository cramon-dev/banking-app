# banking-app
Small banking app for Radancy test.

# install
Configure your system to use Node 14.16.0 and npm 6.14.11.
If using nvm, just run `nvm install 14.16.0`.
Run `npm i`, then `cd` into the banking-app folder and run `ng-serve`.
